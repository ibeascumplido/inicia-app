# INICIA - Sistema de Gestión de Empleados

Aplicación React completa para gestión de vacaciones, presupuestos y usuarios.

## Instalación Rápida

```bash
npm install
npm start
```

Accede a `http://localhost:3000`

**Credenciales:**
- Email: `admin@inicia.com`
- Contraseña: `password`

## Características

✅ Login seguro
✅ Dashboard con métricas
✅ Gestión de presupuestos (CRUD)
✅ Gestión de usuarios
✅ Aprobación de usuarios pendientes
✅ Interfaz responsive
✅ Listo para Vercel

## Deploy en Vercel

1. Sube a GitHub
2. Ve a https://vercel.com
3. Haz clic en "New Project"
4. Selecciona tu repositorio
5. Haz clic en "Deploy"

**¡Listo!** Tu app estará online en 2-3 minutos.

## Acceder

Una vez deployed:
- URL: `https://inicia-app-xxx.vercel.app`
- Email: `admin@inicia.com`
- Contraseña: `password`
